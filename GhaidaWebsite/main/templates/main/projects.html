ABC Sign

The primary goal of the project is to provide an entertaining approach to learning sign language. The project is an interactive application that seamlessly combines education and entertainment.

Initially, the application teaches players how each letter is represented in sign language. Then, players engage in a game where they must press the corresponding key that matches the displayed picture. The final level presents a challenge: players must select the correct signs that represent the letters in various names.

saber: Digital Diagnosis Assistant for Psychologist

The main objective of the saber project is to build an assistant system for the psychologist to facilitate, speed up and standardize the diagnostic process based on the DSM-5 (Diagnostic and Statistical Manual of Mental Disorders).

To overcome the problems reported by psychologists, deep learning algorithm was used in the diagnosis process. In addition, the proposed system assists psychologists by calculating the result of psychological tests in a few seconds. The system was built as a website. CNN algorithm was used with 96% of accuracy to automatically predict the appropriate diagnosis and suggest the most suitable psychological test for the patient take.
